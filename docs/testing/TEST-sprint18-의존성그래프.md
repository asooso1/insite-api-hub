# Sprint 18 테스트 체크리스트 - API 의존성 그래프

**테스트 대상**: API 의존성 그래프 시각화 기능
**테스트 URL**: `http://localhost:3005/?tab=dependencies`
**완료 일자**: 2026-02-03

---

## 1. 기본 기능 테스트

### 1.1 의존성 그래프 탭 접근
- [ ] 사이드바에서 "의존성 그래프" 탭 아이콘이 보이는지 확인
- [ ] 탭 클릭 시 의존성 그래프 페이지로 이동하는지 확인
- [ ] 프로젝트 선택 시 해당 프로젝트의 그래프가 로드되는지 확인

### 1.2 통계 카드 표시
- [ ] 모델 개수 카드가 올바르게 표시되는지 확인
- [ ] 엔드포인트 개수 카드가 올바르게 표시되는지 확인
- [ ] 관계 개수 카드가 올바르게 표시되는지 확인
- [ ] 순환 참조 개수 카드가 올바르게 표시되는지 확인 (있는 경우 빨간색)

### 1.3 핵심 모델 표시
- [ ] "핵심 모델" 섹션에 가장 많이 참조되는 모델들이 표시되는지 확인
- [ ] 참조 횟수가 함께 표시되는지 확인

---

## 2. 그래프 시각화 테스트

### 2.1 노드 렌더링
- [ ] 모델 노드가 보라색 계열로 표시되는지 확인
- [ ] 엔드포인트 노드가 초록색 계열로 표시되는지 확인
- [ ] 노드에 이름과 정보(필드 수, HTTP 메서드)가 표시되는지 확인

### 2.2 엣지 렌더링
- [ ] Request 관계가 녹색 선으로 표시되는지 확인
- [ ] Response 관계가 파란색 선으로 표시되는지 확인
- [ ] Reference 관계가 보라색 선으로 표시되는지 확인
- [ ] 엣지에 화살표가 표시되는지 확인

### 2.3 그래프 조작
- [ ] 마우스 휠로 확대/축소가 되는지 확인
- [ ] 드래그로 그래프 이동이 되는지 확인
- [ ] 노드 드래그로 노드 위치 변경이 되는지 확인
- [ ] 컨트롤 패널의 줌 버튼이 작동하는지 확인
- [ ] "Fit View" 버튼으로 전체 그래프 보기가 되는지 확인

### 2.4 미니맵
- [ ] 미니맵에 전체 그래프 개요가 표시되는지 확인
- [ ] 현재 뷰 위치가 미니맵에 표시되는지 확인

---

## 3. 검색 및 필터 테스트

### 3.1 검색 기능
- [ ] 검색창에 모델명 입력 시 해당 노드만 표시되는지 확인
- [ ] 검색창에 경로 입력 시 해당 엔드포인트만 표시되는지 확인
- [ ] 검색어 삭제 시 전체 노드가 다시 표시되는지 확인

### 3.2 필터 기능
- [ ] "전체" 선택 시 모든 노드가 표시되는지 확인
- [ ] "모델만" 선택 시 모델 노드만 표시되는지 확인
- [ ] "엔드포인트만" 선택 시 엔드포인트 노드만 표시되는지 확인
- [ ] 필터링 시 관련 엣지도 함께 필터링되는지 확인

### 3.3 전체화면 모드
- [ ] 전체화면 버튼 클릭 시 그래프가 전체화면으로 확장되는지 확인
- [ ] 전체화면에서 축소 버튼 클릭 시 원래 크기로 돌아가는지 확인

---

## 4. 영향 분석 테스트

### 4.1 모델 선택
- [ ] 모델 노드 클릭 시 오른쪽에 상세 패널이 나타나는지 확인
- [ ] 선택된 모델의 이름과 필드 수가 표시되는지 확인

### 4.2 영향 분석 결과
- [ ] 영향 수준(Impact Level)이 표시되는지 확인 (low/medium/high/critical)
- [ ] 직접 의존 모델 목록이 표시되는지 확인
- [ ] 간접 의존 모델 목록이 표시되는지 확인
- [ ] 영향받는 엔드포인트 목록이 표시되는지 확인

### 4.3 노드 하이라이트
- [ ] 영향받는 모델 노드가 빨간색으로 강조되는지 확인
- [ ] 영향받는 엔드포인트 노드가 빨간색으로 강조되는지 확인

### 4.4 패널 닫기
- [ ] X 버튼 클릭 시 상세 패널이 닫히는지 확인
- [ ] 빈 영역 클릭 시 상세 패널이 닫히고 하이라이트가 해제되는지 확인

---

## 5. 순환 참조 감지 테스트

### 5.1 순환 참조 경고
- [ ] 순환 참조가 있는 경우 경고 메시지가 표시되는지 확인
- [ ] 순환 참조 경로가 "A → B → A" 형식으로 표시되는지 확인
- [ ] 3개 초과 시 "외 N개" 표시가 되는지 확인

## 6. 다크모드 테스트

### 6.1 다크모드 스타일
- [ ] 다크모드에서 그래프 배경색이 적절하게 변경되는지 확인
- [ ] 다크모드에서 노드 색상이 적절하게 변경되는지 확인
- [ ] 다크모드에서 패널과 카드 색상이 적절하게 변경되는지 확인

- [ ] "Git 저장소를 동기화하여 API를 가져오세요" 메시지가 표시되는지 확인
